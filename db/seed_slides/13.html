Bonus: Anonymous Functions

Functions can take arguments, and these arguments can be functions

window.setInterval() is a function that will call another function on a timer

With a named function:

function timer() { console.log( 'hello' ); } window.setInerval( timer, 1000 ); // will print 'hello' every second ( you've been warned )

An anonymous function has no name, and can be written right into the function call:

window.setInterval( function() { console.log( 'hello' ); }, 1000 );
